HTML tags convey semantic meaning. The strong tag typically displays in bold text, but we do not use the tag because we want to make a word bold, we use this tag to convey strong importance to the word or words. Similarly, the em tag is used to stress emphasis. While the text may display as italics, this tag gives the text the semantics to stress emphasis, not just make text italics.
